
WuzaBoxx
=================================

### Installation
Benötigte Dateien einbinden

    js/wuzabox.js  
    css/wuzabox.css


### Nuztung

Funktion auf gewünschter Seite ausführen und CSS-Selektor als Parameter übergeben.
    
    <div>
        <img src="*" class="enlarge">
    </div>
    
    wuzaBox.gallery(enlarge);
